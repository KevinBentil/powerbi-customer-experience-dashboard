Overview
_
This repository contains a Power BI dashboard built to analyze end‑to‑end customer support performance across states, cities, call centers, channels, and call reasons.
The dashboard offers insights into CSAT, NPS, query volumes, SLA adherence, call duration distribution, and sentiment analysis using interactive and user‑friendly visualizations.
Screenshots of the dashboard pages are provided in this repository.
